# 可视化页面装修
基于 ```vue``` + ```ant-design``` + ```vue-draggable``` 实现


#### DEMO:
https://cullenng.github.io/page-design

#### 依赖:  
[Vue.js](https://cn.vuejs.org/)  
[Ant-deisgn](https://antdv.com/)  
[Vue.Draggable](https://github.com/SortableJS/Vue.Draggable)  


# 开始使用

#### NPM安装
```
npm install
```

#### 本地开发命令
```
npm run dev
```

#### 生产环境编译
```
npm run build
```
